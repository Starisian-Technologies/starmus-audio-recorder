enqueue_production_assets() : void

This is a standard script, as the build process resolves all modules. Optionally depends on SPARXSTAR environment checker if available for optimal device detection, but gracefully falls back to internal detection after 2s timeout. Localizes the script with server-side configuration data.

resolve_version() : string

Resolves the asset version number for cache-busting.
