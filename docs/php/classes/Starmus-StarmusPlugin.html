StarmusPlugin.php : 54

Main plugin class (Singleton).

This class is the central controller for the Starmus Audio Recorder plugin. It ensures that all star_components are loaded in the correct order, handles plugin lifecycle hooks (activation, deactivation), and manages custom capabilities.
