\Starisian\Sparxstar\Starmus\services StarPrivateSlugPrefix

Star Private Slug Prefix Service

Automatically prefixes post slugs with "star-" for posts marked as restricted. This provides a naming convention to identify private/restricted content by URL slug.

star_prefix_slug()

star_prefix_slug(string  $slug, int  $post_id) : string

Apply star- prefix to post slugs for restricted content

Filters the unique post slug generation to automatically prepend "star-" to posts that have the _star_restricted meta key set to '1'. Only applies the prefix if it's not already present.
