StarmusAudioRecorder.php : 56

Main plugin bootstrapper.

Responsibilities:

  • Initialize settings early.
  • Instantiate/wire dependent components (Admin, Assets, UI, REST).
  • Register global hooks once.
  • Defer heavy logic to dedicated classes.
