\Starisian\Sparxstar\Starmus\services StarmusR2DirectService

Direct S3-Compatible (R2/AWS) Audio Service

Minimal SDK-based S3/R2 control for bandwidth optimization versions. Bypasses WordPress plugins for direct storage management of optimized assets. Supports both Cloudflare R2 (primary) and AWS S3 (backup/alternative).
